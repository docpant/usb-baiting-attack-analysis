# Risk Analysis

USB baiting attacks may involve malicious software such as ransomware, spyware, trojans, worms, or remote access malware hidden on removable media devices. If another employee had connected the infected USB drive to a hospital workstation, the malware could have compromised sensitive systems or provided attackers with unauthorized network access.

The USB drive also exposed sensitive personal and organizational information that could be used for phishing attacks, identity theft, or targeted social engineering campaigns. Technical controls such as endpoint protection, USB device restrictions, antivirus scanning, and network monitoring can help reduce these risks.

Operational and managerial controls should include employee cybersecurity awareness training, removable media handling policies, least privilege access controls, and procedures requiring suspicious devices to be investigated inside isolated virtual environments before use.
